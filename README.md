## Topik Aplikasi
Aplikasi **Katalog Bahasa Pemrograman** adalah platform edukasi sederhana yang memungkinkan pengguna untuk:
1. Menjelajahi daftar bahasa pemrograman populer.
2. Mencari bahasa pemrograman tertentu berdasarkan nama.
3. Mengurutkan daftar bahasa secara alfabetis (A-Z dan Z-A).
4. Melihat informasi pada setiap bahasa (Logo, Deskripsi, dan contoh kode "Hello World").
